Implementation

Implementation

.

Our focus is to make our pooled testing techniques as easy and error-proof as possible for lab staff, and we will add an easy-to-use GUI-based tool to capture/guide their work, as well as record correctness and help effectively capture results. In this sense, the required product is a workflow assistant that supports lab technicians throughout all steps of the process. Ideally, we would like this assistance to begin in the protocol design phase, that is, our existing algorithms based on assisting researchers to design an optimal pooling strategy, and then embody this protocol in laboratory tools, with tracking samples as protocol implementations.

Envisioned Solution

The solution we envision is a cross-platform mobile app (Android or iOS) that will allow technicians to run on a smartphone or tablet. We'll use cross-platform frameworks like Lonic and React Native. Generate ios versions of the same functionality from the same Android code base.

Technologies

Version Control: We will use GitHub for version control. We will have master contain functional code, with each task in it's own branch. Commits need to have concise and descriptive titles.

Issue Tracking: GitHub Issues for issue tracking.
